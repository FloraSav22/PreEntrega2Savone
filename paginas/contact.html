Sobre Nosotros

¿Quienes Somos?

Soy una persona aficionada a la Astronomia y la Cienca quien con mucho tiempo libre decidio crear un Catalogo de facil acceso para que todo el mundo pueda saber los ultimos avances de la ciencia y descubrimientos Exoplanetarios.

¿Como empezo el Proyecto ATLAS?

El Proyecto ATLAS comenzó como una idea de crear el catalogo más amplio de planetas y estrellas para el dominio publico, asimismo, gracias al curso de desarrollo web en CoderHouse pude aprender a usar las herramientas que brinda Html, Css, Bootstrap y demás para poder crear el Sitio que estás viendo.

¿Tienes más Preguntas?
